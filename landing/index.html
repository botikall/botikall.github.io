Вебсистема оформлення замовлень кондитерських виробів

Проєкт бакалаврської роботи — Танченко Артем

Опис проєкту

Сучасний ринок кондитерських виробів стає дедалі більш конкурентним. Цей проєкт покликаний автоматизувати процес замовлень і полегшити адміністрування для кондитерських підприємств. Розроблено вебдодаток на базі Django та PostgreSQL.

Основний функціонал

  • Перегляд каталогу продукції з фото та описом
  • Індивідуальне замовлення з додатковими опціями
  • Розрахунок вартості
  • Онлайн-оплата (Stripe/Apple Pay/Google Pay)
  • Відстеження статусу
  • Особистий кабінет користувача
  • Панель адміністратора: управління товарами, користувачами та аналітика

Технології

  • Python + Django
  • PostgreSQL
  • HTML/CSS/JavaScript (Django templates)
  • JWT авторизація, Django REST Framework

Розгортання проєкту

Локально

git clone https://github.com/botikall/botikall.github.io
cd botikall.github.io
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
http://127.0.0.1:8000/
